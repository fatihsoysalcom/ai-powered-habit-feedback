# AI Powered Habit Feedback

This Python script simulates a simple habit tracker. Users input their daily habit completion status for a specified number of days. After tracking, the script provides "AI-powered" insights and recommendations based on the completion history, demonstrating how a system can offer personalized feedback to encourage habit formation, as described in the article.

## Language

`python`

## How to Run

1. Save the code as `habit_tracker.py`.
2. Open a terminal or command prompt.
3. Run the script using `python habit_tracker.py`.
4. Follow the prompts to input your habit completion status for the simulated days.
